RCCM学習ツール PWA版

【利用方法】
1. このフォルダー内の全ファイルを、同じ階層のままHTTPS対応Webサーバーへアップロードします。
2. iPhone: Safariで開き、共有 →「ホーム画面に追加」。
3. Android: Chromeで開き、表示される「アプリをインストール」を選択。
4. 初回オンライン表示後は、問題・画面・アイコンがキャッシュされ、オフラインで利用できます。

【重要】
- index.htmlだけを直接 file:// で開いた場合、通常の学習機能は動作しますが、Service WorkerによるPWA・オフライン機能は有効になりません。
- PWAにはHTTPS（または開発時のlocalhost）が必要です。
- AI解説は外部API通信が必要なため、オフライン時は利用できません。
- APIキー、学習記録、中断データは各端末のブラウザ内に保存されます。
